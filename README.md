# V2Ray Converter & Manager ⚡

یک ابزار تحت وب بسیار سبک، سریع و کاربردی برای تبدیل، ویرایش و مدیریت پیکربندی‌ها (کانفیگ‌ها) و لینک‌های V2Ray. این سیستم به صورت کاملاً استاتیک طراحی شده و برای اجرا نیازی به هیچ‌گونه سرور (Backend) یا پایگاه داده‌ای ندارد.

🌐 **لینک دسترسی به ابزار (Live Demo):** 👉 [https://smblue07.github.io/my-v2ray-tools/](https://smblue07.github.io/my-v2ray-tools/)

---

## 🌟 ویژگی‌ها و امکانات کلیدی

* 🔄 **تبدیل دوطرفه:**
  * **JSON به لینک:** تبدیل آسان کدهای ساختاریافته JSON به لینک‌های استاندارد کلاینت‌های V2Ray (مانند `vmess://`، `vless://` و...).
  * **لینک به JSON:** مهندسی معکوس لینک‌های کانفیگ و تبدیل آن‌ها به فرمت JSON جهت مشاهده دقیق جزئیات و ویرایش پارامترها.
* 🔐 **مدیریت Base64:**
  * **رمزگشایی (Decode):** استخراج لینک‌های کانفیگ از داخل لینک‌های اشتراک (Subscription) که با Base64 کدگذاری شده‌اند.
  * **رمزنگاری (Encode):** تبدیل خروجی نهایی به فرمت Base64 برای ساخت لینک‌های سابسکریپشن اختصاصی و استاندارد.
* 🌓 **پشتیبانی از حالت تاریک و روشن (Dark/Light Mode):** دارای رابط کاربری منعطف که با یک کلیک بین حالت تاریک و روشن تغییر می‌کند تا در هر شرایط نوری، استفاده از آن راحت باشد.
* 📁 **پشتیبانی از فایل:** امکان بارگذاری مستقیم فایل‌های متنی یا JSON (Load File) بدون نیاز به کپی و پیست کردن.
* ⚡ **خروجی سریع:** قابلیت کپی کردن با یک کلیک یا دانلود فایل خروجی.

## 🔒 امنیت و حریم خصوصی (Privacy)

تمام پردازش‌های این ابزار **۱۰۰٪ در سمت کاربر (Client-side)** و داخل مرورگر شما انجام می‌شود. هیچ‌کدام از کانفیگ‌ها، لینک‌ها یا اطلاعات ورودی شما به هیچ سروری ارسال نمی‌شوند و در هیچ دیتابیسی ذخیره نخواهند شد. بنابراین می‌توانید با خیالی کاملاً آسوده از آن برای مدیریت کانفیگ‌های شخصی خود استفاده کنید.

## 🚀 راهنمای اجرای محلی (Local)

اگر دوست دارید این ابزار را روی سیستم خودتان و بدون نیاز به اینترنت اجرا کنید:

۱. این مخزن را دانلود (Download ZIP) یا کلون (Clone) کنید.
۲. فایل‌ها را از حالت فشرده خارج کنید.
۳. فایل `index.html` را با یکی از مرورگرهای خود (مثل کروم، فایرفاکس و...) باز کنید.
